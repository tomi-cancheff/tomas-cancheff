# 🎮 Portfolio: Tomas Cancheff | Game Designer & Game Developer

Static web portfolio with a Cyberpunk/Neon aesthetic, designed to showcase experience in Game Design, Game Development, rapid prototyping, and technical skills.

## 📋 About the Project

This repository contains the source code for my personal portfolio. The main goal is to present my professional profile as a Game Designer and Developer through an immersive and gamified interface.

The site features a modular architecture, optimized for performance, and is natively configured to be deployed on static hosting services like GitHub Pages.

## ✨ Key Features

* **Responsive Design:** Fully adapted for mobile, tablets, and desktop devices.
* **Gamer Aesthetic:** Neon/Cyberpunk visual theme featuring custom CSS animations (glitch, scanlines, CRT effects).
* **Bilingual System:** Real-time English/Spanish language toggle powered by vanilla JavaScript.
* **Interactivity:**
    * Project carousel ("Side Quests") with horizontal scrolling.
    * Functional contact form utilizing the `mailto` protocol.
    * Dynamic canvas background animations (Interactive retro grid with performance toggling).
    * "HP Bar" system acting as a dynamic scroll progress indicator.

## 🛠️ Technologies Used

The project follows a strict Separation of Concerns (SoC) architecture:

* **HTML5 (52.8%):** Semantic structure and content layout (`index.html`).
* **CSS3 & Tailwind CSS (29.5%):** Custom styling, keyframe animations, and utility-first responsive design (`css/styles.css`).
* **Vanilla JavaScript (17.7%):** Logic for localization (i18n), canvas rendering, debounced scroll events, and DOM manipulation (`js/main.js`).
* **External Assets:** Font Awesome (Iconography) and Google Fonts (*Press Start 2P*, *Share Tech Mono*).
